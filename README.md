# 1. 前言
金融领域，尤其是证券行业，获取专业资格是至关重要的一步。

SIE 考试的目的不仅仅是为了获取专业资格，更重要的是通过准备过程，深入理解金融和证券体系，从而助力我们的日常工作或个人投资理财。

尽管人工智能工具日益成熟，可以帮助我们查找知识细节，但是我们需要先有对知识体系整体的视野和理解，才能更好的应用于实际。

本文将基于提供的书籍章节内容，详细解读 SIE 考试的各个环节，帮助你做好准备，以最有效的方式通过考试。

# 2. 简介
SIE（Securities Industry Essentials）考试是由美国金融业监管局（FINRA）开发的金融行业入门考试，旨在帮助考生掌握金融领域的基本知识及证券行业的基本概念。

1. 入门性（Essentials）

SIE 是证券行业的“基础”考试，旨在测试对金融市场基本概念和规则的了解，而不是特定产品或销售技巧。

2. 前置性（Prerequisite）

SIE 本身并不能让你从事任何具体的证券业务。通过 SIE 后，你还需要通过一门代表资格考试（Representative Level Exam），如 Series 6、Series 7 等，才能成为注册代表并从事相关证券业务。

3. 无赞助要求（No Sponsorship Required）

这是 SIE 最关键的特点之一。任何人（包括大学生、转行人士、对金融感兴趣的人）都可以报名参加，不需要由券商（Broker-Dealer）赞助。这为人们进入证券行业提供了一个低门槛的“敲门砖”。

4. 成绩有效期

通过 SIE 后，成绩有效期为4年。你需要在4年内找到一家愿意赞助你的券商，并通过所需的代表资格考试，才能完成注册。

通过 SIE 后，可以转入多种金融工作，如投行、证券分析、私募发行、行业调研和证券交易等，不过前提是还要考取具体的从业执照。

# 4. 注意事项
1. 考试时间窗口

考试时间友好，有北京时间白天的考点，建议提前两周以上预约，选择合适的时间，否则抢不到。

2. 了解答题界面

可以在线体验考试答题页面，[点击这里](https://www.finra.org/sites/default/files/external_apps/proctor_tutorial.swf.html)。

3. 线上考试环境要求

需要独自在封闭安静的环境中进行考试，环境检查非常严格，桌子周围不允许有其他物品等等（详细要求见邮件指引）。
- 不能戴珠宝首饰、手表；
- 衣服口袋必须清空，不能留任何物品；
- 房间里不能纸笔、显示屏等电子设备；
- 桌面保持干净整洁，不能有其他与考试无关的东西；
- 绝对不能让其他人进入房间。

会有线上环境检查官说英文引导并详细检查周围环境，通过后，会有线上考官引导进入正式考试流程，听不清可以要求对方重述或输入信息到线上对话框中。
- Eg: Could you please repeat that? I didn't hear clearly.
- Eg: Could you please type that instruction in our chat window?

正式考试会有 75 + 5 = 80 道题目（5道不算分，但是我们不知道是哪5道不算）。考试时长1小时45分钟，一股气答下来还是有些累的。中间可以要求休息10分钟但是人要保持在摄像头范围内。

考试过程中无需和考官对话，只需在对话框中输入 I would like to take a 10-minute break. 然后按照考官指示和要求休息。

# 5. 备考流程
1. 了解 SIE 大纲

https://www.finra.org/sites/default/files/SIE_Content_Outline.pdf

可以将上述大纲输入 AI，让其生成中文脑图，便于快速了解。

2. 知识点掌握

以书籍《SIE Exam 2025-2026 For Dummies》的内容来学习。

该书籍暂未找到在线版本，可以在 [dokumen.pub](https://dokumen.pub/sie-exam-2025-2026-for-dummies-securities-industry-essentials-exam-prep-practice-tests-amp-flashcards-online-9781394261871-9781394261888-9781394261895.html) 下载 PDF 版本。

书中前四章为介绍 SIE 概况，可阅读找找英文的感觉。

详细阅读每个章节的知识点，并利用 AI 工具弄清楚不明白的知识点。

此外，**本人参考 Dummies 一书整理了各个章节的核心知识点，做了适当的精简及补充**，可辅助参考。

3. 章节练习

学习完每一章节的知识点后，务必完成书中对应章节的练习题，以加深对知识的理解、熟悉英文题目。

4. 题库练习

完成《SIE Exam 1001 Practice Questions For Dummies》一书中的练习题，查缺补漏，加深对知识点的掌握，并了解考试的出题模式。

提一嘴，关于 1001 这个数字不仅表示书籍有 1001 道题目，更是强调其提供了海量且全面的练习题。1001 源自阿拉伯民间故事集 《一千零一夜》（Arabian Nights） 。在这本书里，聪明的谢赫拉莎德用一千零一个夜晚的故事吸引了国王。从此，1001 在西方文化中就带上了 “数量极多、内容丰富、包罗万象” 的象征意义。

对应的，西方 101 代表 “从零开始的基础知识”。这个用法源于北美教育体系中的课程编码。在大学里，编号为101的课程（如 Psychology 101, Economics 101）通常是该学科的第一门、最基础的导论课，面向没有任何背景知识的新生。

该书籍暂未找到在线版本，但可在 [dokumen.pub](https://dokumen.pub/sie-exam-1001-practice-questions-for-dummies-for-dummies-business-amp-personal-finance-9781394195244-9781394195251-9781394195268-1394195249.html) 下载 PDF 版本。

5. 模拟考试

按顺序进行四套模拟题的练习，控制时间以模拟真实考试环境，并检测自己的分数：
- Dummies 书中最后两套题。
- Knopman 网站提供的一套题（Knopman 是华尔街和金融执照备考领域内非常知名和受人尊敬的培训机构）。
- FINRA 官网提供的一套题。

通过以上方法，可以系统了解 SIE 考试内容，为顺利通过考试打下坚实基础。

# 7. FAQ
1. 考试报名费用多少？

80 美元。

2. 多少分算通过？

SIE 及格分数为70分（分值从0到100），也就是要答对 70% 的题目。

4. SIE 未通过重考间隔时间？

SIE 考试的重考政策（Cooling-off Period）旨在让你有足够的时间进行额外准备。以下是核心的重考规则：
考试尝试|等待间隔|备注
:---|:---|:---
第一次重考（即第二次考试）|无需等待|可以在第一次考试失败后立即预约第二次考试。
第二次重考（即第三次考试）| 30天 | 从第二次考试失败之日起，需等待30个自然日才能预约第三次考试。
后续所有重考（第四次及以上）| 180天 |从最近一次考试失败之日起，每次失败后都需要等待180个自然日才能进行下一次考试预约。

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/4f4c32b87dd24d788a6228c3237631f5.png)

5. 考试预约成功后，可以修改或取消预约吗？

可以修改或取消考试预约，不过可能会收费。

- 提前 10 个工作日以上免费。
- 提前 3-10 个工作日收取考试报名费的一半。
- 若在考试日期前 2 个工作日内改期 / 取消，或考试当天 “未到场”“迟到无法入场”，需承担 “全额考试费” 的损失。

详见 [Reschedule or Cancel Your Appointment](https://www.finra.org/registration-exams-ce/qualification-exams/cancellation-policy)。
